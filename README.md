# zomato-data-analysis
This interactive Power BI dashboard provides a comprehensive analysis of restaurant data in Bangalore. It helps identify customer trends, restaurant performance, and key business metrics to support data-driven decision-making.

Business Problem
Zomato wants to understand:-
1.Which restaurant categories drive maximum engagement?
2.Do online ordering and table booking impact customer votes?
3.What pricing range dominates Bangalore’s restaurant market?
4.Which areas show highest customer activity?
This analysis helps restaurant owners and aggregators optimize pricing, service offerings, and expansion strategy.
